# 뱀서라이크 같은 게임
대한상공회의소 유니티 부트캠프에서 페어 프로그래밍 프로젝트로 제작한 게임입니다.  
뱀서라이크 장르 상 오브젝트 풀링, 컴포지트 패턴과 같은 시스템 원리를 적용하는 예제로 개발하였습니다.

## 1. 게임 장르
- 뱀서라이크, 로그라이크
## 2. 담당 역할
- 2인 팀 프로젝트
  (개발자 2명)

- 구현 내용
1) 캐릭터 시스템
2) 타겟 시스템
3) 아이템 시스템

## 3. 개발 기간
- 2023.12 - 2024.01 (1주일)
## 4. 개발 환경
- Windows 11, Unity Editor 2022.3.11f1, VisualStudio 2022 - 17.7.6, Text Mesh Pro 3.0.6, JsonSerialize 1.0.0
